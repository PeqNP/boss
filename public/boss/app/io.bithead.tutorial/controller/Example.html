UIListBox: A sortable list box where options may be re-ordered in the list. The API also allows the consumer to "accept" a position before committing the option to its new position. Such that, if a network request is required, before the option is moved to the new position, the consumer can pass in a function, that returns a Promise (e.g. async function). If the Promise throws an Error, the position will not be updated.
