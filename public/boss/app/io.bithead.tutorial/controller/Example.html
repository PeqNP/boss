UIListBox: List box where only one option may be selected.

UIListBox: List of options, where multiple may be selected at once.

UIListBox: List of options where each option is treated as a button.

UIListBox: A sortable list box where options may be re-ordered in the list. The API also allows the consumer to "accept" a position before committing the option to its new position. Such that, if a network request is required, before the option is moved to the new position, the consumer can pass in a funciton, that returns a Promise (e.g. async function), it will call that function. If the Promise throws an Error the position will not be updated.

UIPopupMenu: Pop-up menu shows a list of options.
